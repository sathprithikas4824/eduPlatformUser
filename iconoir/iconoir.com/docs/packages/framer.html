Iconoir - Framer

Iconoir is happily part of Framer.

To start using the icons: On the top menu, Insert > Graphics > Iconoir.

You can switch between icons from the right sidebar in the editor.

More?
If you would like to use Iconoir on a different library or if you want to contribute to support a new library, open a feature request in our official GitHub repository.
